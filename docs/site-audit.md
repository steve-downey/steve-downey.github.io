## 1. Broken Internal Links (194 issues)

**Severity: BROKEN**

**Root cause:** All 97 `REDIRECTIONS` in `conf.py` point to HTML file paths (e.g.,
`/posts/index.php/2004/09/28/tbr-agyar-by-steven-brust.html`) but Nikola's
output structure uses directory-based URLs with `index.html` inside (e.g.,
`/posts/index.php/2004/09/28/tbr-agyar-by-steven-brust/index.html`).

The redirect generator in `conf.py` produces meta-refresh HTML that contains
these broken paths as targets.

**Affected pages:** All 97 redirect stubs in `output/` (each checked twice by
nikola check -l = 194 total warnings).

**Example:**
```html
<!-- Source: /output/index.php/2004/09/28/tbr-agyar-by-steven-brust/index.html -->
<meta http-equiv="refresh" content="0; url=/posts/index.php/2004/09/28/tbr-agyar-by-steven-brust.html">
<!-- This path does not exist in output/ -->
```

**Actual path in output:**
```
/index.php/2004/09/28/tbr-agyar-by-steven-brust/index.html
```

**Affected redirect entries in conf.py (all 97):**
- Pages: `/pages/index.php/2020/11/28/twitter/` → target `/pages/index.php/2020/11/28/twitter.html` ✗
- Pages: `/pages/index.php/2020/11/28/github/` → target `/pages/index.php/2020/11/28/github.html` ✗
- Pages: `/pages/index.php/about/` → target `/pages/index.php/about.html` ✗
- Posts: all 94 post redirects follow the same pattern

**Status:** This is a systematic configuration issue, not individual broken links.

## 5. External CDN Dependencies

**Severity: CONCERN (technical debt)**

The site loads external resources from CDNs with no local fallback:

### 5.1 Tailwind CSS Play CDN (HIGH PRIORITY)
- **URL:** `https://cdn.tailwindcss.com?plugins=typography`
- **Loaded:** Always, on every page
- **Issue:** 
  - Requires internet connectivity
  - Slow (external request on every page load)
  - Not suitable for production static site
  - Sends unique request each page visit (no caching benefit)
- **Source:** `themes/sdowney-tailwind/templates/base_helper.tmpl` line ~10
- **Fix:** Build CSS artifacts locally (JIT CSS or pre-built Tailwind output)

### 5.2 Google Fonts (MEDIUM PRIORITY)
- **URLs:** 
  - Preconnect: `https://fonts.googleapis.com`, `https://fonts.gstatic.com`
  - Fonts: Open Sans, Source Code Pro, Source Sans 3
- **Loaded:** Always
- **Issue:** External, privacy implication (font requests expose page visits), no
  local fallback
- **Source:** `base_helper.tmpl`
- **Fix:** Host fonts locally or use system fonts

### 5.3 Font Awesome 7.0.1 Icons (LOW PRIORITY)
- **URL:** `https://cdnjs.cloudflare.com/ajax/libs/font-awesome/7.0.1/css/all.min.css`
- **Loaded:** Always
- **Issue:** External CDN, but has SVG/local fallback option, lower impact
- **Usage:** Sidebar theme toggle uses `fa-solid fa-circle-half-stroke`
- **Source:** `base_helper.tmpl`
- **Fix:** Use SVG inline or switch to system icons

## Recommendations (Priority Order)

### P0 — Fix REDIRECTIONS Configuration (BLOCKER)
All 97 redirects are non-functional. The target paths need to match Nikola's
directory-based URL structure:
```python
# Current (broken):
["source", "/posts/index.php/2004/09/28/tbr-agyar-by-steven-brust.html"]

# Should be:
["source", "/posts/index.php/2004/09/28/tbr-agyar-by-steven-brust/"]
# OR:
["source", "/posts/index.php/2004/09/28/tbr-agyar-by-steven-brust/index.html"]
```
After fixing, all 194 link check warnings will resolve.

### P1 — Address wp-content References (EXPECTED, NOT FIXABLE)
The 2015 and 2019 posts cannot be fully fixed without original image files.
Options:
- Accept as-is (reasonable — no one controls these old posts)
- Remove non-functional image tags from source
- Find/host original images elsewhere

### P2 — Migrate Tailwind CSS to Self-Hosted (TECHNICAL DEBT)
Build a production CSS bundle instead of loading Play CDN on every page. This
is a build-time concern, not a runtime blocker.

### P3 — Self-Host Fonts (OPTIONAL)
Lower priority than Tailwind CSS. Consider privacy implications if this is
relevant to your audience.
